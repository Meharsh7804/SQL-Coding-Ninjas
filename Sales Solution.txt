Problem statement :-
Given three tables: salesperson, company, orders.
Output all the names in the table salesperson, who didnâ€™t have sales to company 'RED'.

Solution :-
SELECT name
FROM Salesperson
WHERE sales_id NOT IN (
    SELECT sales_id
    FROM Orders
    WHERE com_id = (
        SELECT com_id FROM Company WHERE name = 'RED'
    )
);
